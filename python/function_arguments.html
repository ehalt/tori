Let's learn about function arguments...

Default argument values

def banner(message, border = '_'):

এই ফাংশনে দুটি Arguments আছে। প্রথমটি message, এবং দ্বিতীয়টি হচ্ছে border এখানে border আর্গুমেন্টে একটি ডিফল্ট ভ্যালু সেট করা আছে যা ‘ _ ‘

Note: Arguments with default values must come after those without default values. 

                              

>>> def banner(message, border='_'):
line = border * len(message)
print(line)
print(message)
print(line)


>>> banner("Torikus Sadik")
_____________
Torikus Sadik

নিচের প্রোগ্রামে একটি মজার বিষয় খেয়াল করিঃ 


                              

>>> def add_spam(menu=[]):
menu.append("Spam")
return menu

>>> add_spam()
['Spam']
>>> add_spam()
['Spam', 'Spam']
>>> add_spam()
['Spam', 'Spam', 'Spam']

এই প্রোগ্রামে যতবার add_spam() কে কল করার হয়েছে লিস্ট তত বড় হয়ে চলেছে। এই প্রবলেমের সলিউশন এবার জানা জাক।

immutable default value 

                              

>>> def add_spam(menu = None):
if menu is None:
        menu = []
menu.append('spam')
return menu
