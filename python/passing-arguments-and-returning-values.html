Passing argument & returning values

Let's learn this topic

Argument Passing: 

                              

>>> m = [9, 15, 24]
>>> def modify(k):
        k.append(39)
        print("k =", k)

        
>>> modify(m)
k = [9, 15, 24, 39]
>>> m
[9, 15, 24, 39]

Argument passing semantics


Replacing argument value 

                              

>>> f = [14, 23, 37]
>>> def replace(g):
        g = [17, 28, 45]
        print("g = ", g)

        
>>> replace(f)
g =  [17, 28, 45]
>>> f
[14, 23, 37]


Mutable Arguments 


                              

>>> def replace_contents(g):
g[0] = 17
g[1] = 28
g[2] = 45
print("g = ", g)


>>> f = [14, 23, 37]
>>> replace_contents(f)
g =  [17, 28, 45]
>>> f
[17, 28, 45]

Return Semantics 

                              

>>> def f(d):
return d

>>> c = [ 6, 10, 16 ]
>>> e = f(c)
>>> c is e
True
