Tuple

Let's learn about Tuple in detail...

Tuple Example program: 

                              
>>> t = ('tori', 4.9, 3)
>>> t
('tori', 4.9, 3)

Indexing: 

                              

>>> t = ('tori', 4.9, 3)
>>> t
('tori', 4.9, 3)
>>> t[0]
'tori'
>>> t[1]
4.9
>>> t[2]
3

Loop: 

                              

>>> t = ('tori', 4.2, 3.2, 1)
>>> for items in t:
        print(items)

        
tori
4.2
3.2
1

To know type: 

                              
                                  >>>type(t)

Tuple unpacking: 

                              

>>> def minmax(items):
return min(items), max(items)

>>> minmax([83, 33, 84, 32, 85, 31, 66])
(31, 85)
>>> lower,upper = minmax([83, 33, 32, 85, 31, 86])
>>> lower
31
>>> upper
86
