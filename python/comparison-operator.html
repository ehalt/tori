Comparison operator

Let's learn about comparison operators...

এ পর্বে আমরা জানব কিছু তুলনা মূলক অপারেটর সম্পর্কে। তাহলে শুরু করি চলো,

Equal to:

এই অপারেটর সম্পর্কে তোমাকে বলার আগে আমি একটি প্রোগ্রাম লিখি তাহলে তোমার বোঝার সুবিধে হবে। 

                              
>>> 42 == 42
True

উপরের প্রোগ্রামে খেয়াল করো, ৪২ ইকুয়াল টু ৪২। দুটা নাম্বার সমান হওয়াতে আউটপুট True প্রদর্শিত হয়েছে। এবার তোমাকে একটা কাজ দেই। তুমি 42 এর স্থানে 43 লিখে প্রোগ্রাম রান করো। তাহলে পার্থক্য বুঝতে পারবে।


Not equal to: 


>>> 3 != 3
False

এই প্রোগ্রামের প্রথম লাইনে বলা হয়েছে প্রথম ৩ দ্বিতীয় ৩ এর সমান নয়। অথচ দুটো তিন ই সমান। তাই প্রোগ্রামের আউটপুট False আসছে। এবার আরেকটি উদাহরণ দেখা যাকঃ 

                              
>>> 2 != 3
True

এখানে উপরের প্রোগ্রামে 2 ছোট এবং 3 বড়। তাই প্রোগ্রাম টি সত্য।


Less than: 

                              
>>> 45 < 50
True

উপরের প্রোগ্রামে ৪৫ ছোট এবং ৫০ বড় তাই প্রোগ্রাম টি সঠিক। প্রোগ্রাম টি সঠিক হওয়াতে আমরা True আউটপুট পেয়েছি।


Greater than 

                              
>>> 50 > 45
True

উপরের প্রোগ্রামে ৫০ বড় এবং ৪৫ ছোট হওয়ায় প্রোগ্রামটি সঠিক এবং তাই True প্রিন্ট হয়েছে।


Less than or equal to: 

                              
>>> eggprice = 32
>>> eggprice <= 32
True

উপরের প্রোগ্রামে প্রথম লাইনে আমরা একটি ভেরিয়েবল নিয়েছি যেখানে ডিমের দাম সেট করে দিয়েছি ৩২ টাকা। এবং দ্বীতিয় লাইনে বলে দিয়েছি ডিমের দাম ৩২ এর নিচে অথবা ৩২ এর সমান।


Greater than or equal to 

                              
>>> age = 20
>>> age >= 15
True

উপরের প্রোগ্রামে প্রথম লাইনে আমরা একটি ভেরিয়েবল নিয়েছি যেখানে আমার বয়স সেট করেছি ২০ বছর। পরের লাইনে বলে দিলাম যে আমার বয়স ২০ অথবা ২০ এর সমান।
